About Me

I am Rajpreet, a master's student pursuing HPC at TU Munich, Germany. My research spans Programming Language for AI Agents, Low-Latency Retrieval Systems, High Performance Computing (C++, CUDA, Rust), FPGA based Digital Design of Accelerators, Complexity Theory, Type and Category Theory.

My master's thesis consists of developing a programming language for AI agents. I was awarded institute's silver medal for best performance in the capstone. I really try to make a balance between Abstract Mathematics, Programming, and Applied Mathematics and enjoy the interdisciplinary paradigm.

Programming Language for AI Agents

[Master's Thesis] The design and implementation of a programming language (named .sl) specifically for AI agent development and orchestration. It addresses the fundamental disconnect between high-level agent behavior specification and low-level infrastructure management by providing native abstractions for agentic workflows, inter-agent communication, and distribution of workloads.

Multi-Agent System for Traceability and Requirements Analysis

Designed and developed a multi-agent AI system to automate and optimize System Architecture and Requirements processes. Built a scalable PostgreSQL backed data layer with vector embeddings and integrated GraphRAG pipelines to enable structure-aware retrieval. Applied DPO for agents alignment, built evaluation suites for performance and reliability assessment.

Internship in SARE Team at Qualcomm

High Performance Retrieval System and Agentic AI

Developed and filed a patent for a state-of-the-art retrieval system (CSR-RAG) used in Natural Language to SQL query generation. Also invented a new Hypergraph ranking algorithm solving an NP-hard optimization problem of resource allocation approximately. Deployed the full stack agentic pipeline for NL2SQL workflow, utilizing CSR-RAG for high-performance.

Research Internship in Autonomous Systems Group at Bell Labs

First Underwater Robots to clean the oceans

Built a C++ application for Real-Time Depth Estimation with 4 framos cameras to run on NVIDIA Jetson Orin. The system uses stereo vision and computer vision algorithms to generate depth maps of underwater environments. This enables autonomous navigation and obstacle avoidance for underwater robots designed to clean oceans. I optimized the C++ application with SIMD and multi-threading to achieve real-time performance of 25 frames/second. [Project SeaClear]

Graduate Student Research Assistant at the Chair of Information Oriented Control, TU Munich

Current Topics

I am working on various topics at the moment. Some of them are going for publications, and others are for development purposes.

Fully Homomorphic Encryption on FPGAs

This is my independent researh work on Homomorphic Encryption. I am exploring the implementation of Fully Homomorphic Encryption on FPGAs with increased main memory bandwidth or its improved utilization. There is another good direction to use in-memory/near-memory computing. My ideas are inspired from the works of Prof. Vinod Vaikuntanathan from M.I.T. and Prof. Onur Mutlu from Safari Research Group at E.T.H. Zurich. If you want to discuss this in detail, please send me an email.

A.I. based Operating System

I am trying to incept the idea of Artificial Intelligence based Real-Time Operating System. The language used for this development is C++. The idea for dynamic operating system/conditions are very natural and obviously require a solution. This can give advantage in Space Exploration, Meta-learning, Robotics, etc. The working is not simple and might require complex analysis of the fundamental logic learning theory. At the top-most abstraction layer, it would work interchangeably with text, speech, videos, etc. And will be easily able to communicate, explain, generate intelligent responses. This could be a nice application of LLMs to low-level digital logic. Let's see how it goes, I am open for discussions.

Digital Hardware Modeling using Verilog

This topic deals with a Pipeline implementation of a processor in Verilog. The processor is really basic, but my vision is to make a design of an "FPGA" i.e. going one step beyond the usual hardware design process.

C++ Application for Algorithmic Adjoint Differentiation

Writing a Low-Latency C++ software for Algorithmic Adjoint Differentiation in Finance Applications. This is an exciting application of modern C++ concepts to Quantitative Finance. Please feel free to check out the Github repo for this project.

Machine Learning with RUST & Graphics Programming in WebGL

I am exploring the implementation of Machine Learning Algorithms in RUST and comparing its performance for fast and safe code. Rust is well suited for Systems Programming, but it can be a good option for HPC tasks as well.

Computational Algebraic Geometry

I am following Prof. Emre Sertoz for his work in Computational Algebraic Geometry. This areas is really fascinating for me personally, and I am taking some time pick a right topic for publication.

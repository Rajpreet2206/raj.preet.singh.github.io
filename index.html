Convolutional AutoEncoders on FPGAs

This is a guided researh work on implementing an optimized design of "Convolutional AutoEncoders" on FPGA for image reconstruction. This involves RTL design, scheduling, allocation, binding, and data-path generation of this deep learning based approach using Verilog. I am guided by two two Phd students of Dr. Martin Schulz at the Chair of Computer Architecture and Parallel Systems, TU Munich.

Rust based Hypergraph Processing

This is a hands-on seminar work on "Large Scale Graph Processing" and it deals with designing a RUST based hypergraph processing system for distributed systems. It has already outperformed Julian Shun's Hygra framework as well as Hyper. The entire framework is build using RUST which provides performance comparable to C++, and gives memory safety with amazing support of concurrency. Presentation

Real Time C++ Application for an Underwater Robot

I have been associated with project SeaClear since January 2023 as a "Graduate Research Student Assistant". I have mainly focused on the Computer Vision part of the robot while building the depth-estimation solution with Framos module system and NVIDIA Jetson Orin platform.

Compiler: Adding a new backend to LLVM

Implementing a compiler using LLVM, generating intermediate representation (IR). Also adding optimizations to compile the IR to high performance machine code using modern C++. I also aim to add a new back-end to LLVM soon.

Performance Engineering on Supercomputers

I have successfully completed two Master's Internships, i.e.BEAST and Efficient Programming of Supercomputers. Mainly dealing with running performance benchmarks, identifying bottlenecks, cache locality, memory access, programming with SIMD intrinsics, CUDA, OpenMP and MPI.

DevOps: Working student in Munich

I have developed a geo-spatial Flask application the deals with analyzing natural perils like Wildfire, Earthquakes and Tropical Cyclones. Also deployed it on a Kubernetes cluster while optimizing the entire deployment pipeline by leveraging Groovy scripts and Helm for seamless build automation.
